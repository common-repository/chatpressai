ChatPress is a simple solution for integrating AI research and blogging into WordPress

== Description ==

__Easily integrate the OpenAI API to create a research and blogging tool directly in WordPress__

ChatPress is a WordPress plugin that allows users to ask for information, text, ideas etc. and receive a response directly in the post or page edit screen.  To do this the plugin will access ChatGPT’s artificial intelligence capabilities (API) to ask questions, generate content ideas and more.  You can then simply add the content as a headline, post or excerpt at the click of a button.

It’s as simple as chatting to a friend or mentor – ask a question and the plugin will reply!

With ChatPress, you can quickly and easily access OpenAI’s incredible artificial intelligence capabilities (that power the ChatGPT app) to get answers to your questions, generate content ideas, and more. Simply type in your question and hit send, and OpenAI’s API will provide you with a helpful response in just seconds.

Whether you’re a blogger, content creator, or business owner, ChatPress can help you save time and streamline your workflow by providing instant access to ChatGPT’s AI-powered insights and suggestions

This plugin includes -

= ChatPress Features  =

* Adds an AI query block on a Page or Post edit screen
* Simple integration of the OpenAI API


= ChatPress Premium Version =
* Automatic copy/paste to Title, Edit and Excerpt fields
* Append new answers in the Edit field
* Seperate query screen accessible from sidebar (useful if you don't want to open a new Page or Post
* Removes answer word limit of 200 words in free version

== Installation ==

###Manual Install From WordPress Dashboard

If your server is not connected to the Internet, then you can use this method-

1. Download the plugin by clicking on the blue 'Download' button above. A ZIP file will be downloaded.
2. Login to your site’s admin panel and navigate to Plugins -> Add New -> Upload.
3. Click 'Choose File', select the plugin file and click install.

###Install Using FTP

If you are unable to use any of the methods due to internet connectivity and file permission issues, then you can use this method-

1. Download the plugin by clicking on the blue 'Download' button above. A ZIP file will be downloaded.
2. Unzip the file.
3. Launch your favourite FTP client. Such as FileZilla, FireFTP, CyberDuck etc. If you are a more advanced user, then you can use SSH too.
4. Upload the folder to wp-content/plugins/
5. Log in to your WordPress dashboard.
6. Navigate to Plugins -> Installed
7. Activate the plugin.
